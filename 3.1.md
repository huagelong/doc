   模板引擎
========

[参考](https://laravel-china.org/docs/laravel/5.6/blade/1375)

####布局标签 

```
@yield('title')

@section('page-bar')
@endsection

@extends('layouts.app')
```
####变量输出

`{{ $slot }}`

####显示未转义数据

`Hello, {!! $name !!}.`

####路由名称对应的网址

* @url(route_name,params?)
 返回路由名称对应的网址
 ```
     @url("test") //return '/'
     @url("test", ["p"=>1]) //return '/?p=1'
 ```
* @uri
 ```
  {{ @url("test") }}

   {{-- 等价于 --}}
   @uri("test")

 ```
 
 ####引用 资源
```
    @static('/static/js/xxx.js')
    @static('/static/css/xxx.css')
```

####组件

`@widget("admin_header")`

####If 语句

```
@if (count($records) === 1)
    我有一条记录！
@elseif (count($records) > 1)
    我有好几条记录！
@else
   我没有记录！
@endif

```

####foreach 

```
@foreach ($users as $user)
    <p>此用户为 {{ $user->id }}</p>
@endforeach
```

    //
@endphp
```

```
